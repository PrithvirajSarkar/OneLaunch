# OneLaunch 🚀

Launch your workspace in one click.

## About

OneLaunch is a Windows desktop application built in Java that allows users to create custom workspaces and launch multiple applications with a single click.

## Version 1 Features

- Create Workspace
- Add Applications
- Edit Workspace
- Delete Workspace
- Launch Workspace

## Tech Stack

- Java
- VS Code
- Git
- GitHub
